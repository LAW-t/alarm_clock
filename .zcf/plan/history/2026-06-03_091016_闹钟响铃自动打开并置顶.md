# 闹钟响铃自动打开并置顶

## 问题
闹钟响铃时，AlarmActivity 未能自动弹出，用户需要手动找到应用再关闭闹钟。

## 根因
1. Android 10+ 后台启动 Activity 限制，`AlarmReceiver.startActivity()` 可能静默失败
2. `AlarmService` 的 `fullScreenIntent` 缺少 extras（shift/alarm_id/snooze_remaining）
3. `FLAG_SHOW_WHEN_LOCKED` 已弃用（API 27+）
4. Android 14+ 需声明 `USE_FULL_SCREEN_INTENT` 权限

## 方案
双通道保障：
- 通道 1：`AlarmReceiver.startActivity()`（Android 9- 直启，外层 try-catch 兜底）
- 通道 2：`fullScreenIntent` 通知拉起（Android 10+ 主通道）

## 变更文件
1. `AndroidManifest.xml` — +`USE_FULL_SCREEN_INTENT` 权限
2. `AlarmReceiver.kt` — `startActivity()` 外包 try-catch
3. `AlarmService.kt` — fullScreenIntent 补 extras + SINGLE_TOP
4. `AlarmActivity.kt` — `setShowWhenLocked(true)` + `setTurnScreenOn(true)` 替换弃用 API
